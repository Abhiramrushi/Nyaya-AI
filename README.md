# Nyaya-AI 🚀

Nyaya-AI is an AI-powered platform for legal judgment analysis, document processing, and intelligent insights. Built for hackathon.

## Features
- **Backend**: FastAPI with Python, PostgreSQL integration, document pipeline, intelligence engine.
- **Frontend**: React + Vite for dashboard and judgment viewer.
- **Services**: Document processing and AI intelligence.
- **Docker**: docker-compose.yml for easy setup.

## Quick Start
1. Backend: `cd backend && pip install -r requirements.txt && uvicorn main:app --reload`
2. Frontend: `cd frontend && npm install && npm run dev`
3. Or use `docker-compose up`

## Structure
- `backend/`: API, models, routers.
- `frontend/`: React app.
- `uploads/`: File uploads.
